# 🎬 VidAI Creator

Turn your photos and ideas into stunning videos with AI. Share instantly to TikTok, WhatsApp and more.

## ✨ Features

- **✍️ Text to Video** - Describe your idea and AI brings it to life
- **🖼️ Photos to Video** - Upload your photos and create beautiful montages
- **🎵 Music & Captions** - Add music and captions to make it pop
- **🌍 Translate Video** - Upload a video and translate it to another language

## 🚀 Live Demo

[https://video-creator-frontend.onrender.com](https://video-creator-frontend.onrender.com)

## 🛠️ Technologies Used

- **Frontend**: React, Tailwind CSS, React Router
- **Backend**: Node.js, Express
- **AI APIs**: Replicate, Dreamina-Seedance-2.0, Hugging Face
- **Translation**: LibreTranslate

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/video-creator-app.git

# Install dependencies
npm install

# Start the backend server
node server.js

# Start the frontend (in a new terminal)
npm start
